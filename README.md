# 🏙️ 城市交通大亨 🚗

这是一个基于Vue.js的城市交通经营模拟游戏，包含巴士、飞机、地铁、高铁等多种交通方式！

## ✨ 游戏特色

- 🚌 **真实线路**：基于真实世界的澳门及其他城市公交线路
- ✈️ **多种交通**：巴士、飞机、地铁、高铁、共享单车
- 🏢 **公司经营**：30级公司等级系统
- 👥 **员工管理**：司机、飞行员、空姐、维修人员等
- 💰 **经济系统**：完整的收支记录和离线收益
- 💾 **存档系统**：加密存档，支持导入导出
- 🎨 **精美UI**：现代化界面设计

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

🎮 游戏玩法
1.初始状态：10万资金，2辆巴士1辆长途车，2条巴士线路1条长途线路，3个司机
2.经营发展：创建线路、购买车辆、雇佣员工
3.扩展城市：解锁更多城市，建立交通网络
4.升级系统：升级车辆、设施、服务
5.离线收益：离线也能获得收益，最多一天
📱 功能模块
- 📊 运行状况：实时监控所有交通工具
- 💰 财务报表：详细的收支记录
- 🛒 商店系统：购买车辆、设施、共享单车
- 👥 员工管理：雇佣和管理各类员工
- 🛤️ 线路管理：创建和管理交通线路
- ⚙️ 游戏设置：存档管理、游戏重置

- 原生JavaScript
